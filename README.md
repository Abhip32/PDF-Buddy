# PDF Buddy
A responsive Flask web app used for modifying PDF files.
## Table of contents
* [About](#about)
* [Technologies](#technologies)

## About
* Consists of merging, splitting, encrypting, rotating, adding watermark to, and removing pages from PDF files.

* HTML5
* CSS3
* Bootstrap
* Flask
* Python
* JavaScript
